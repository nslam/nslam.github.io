2019-04-18
随笔
摸鱼, 无动力, 迷茫

其实这篇博客应该是要上礼拜天的时候写的，因为我给自己定的小目标是每周日写一篇博客来记录一下本周做了什么，然而一直拖着拖着就拖到了现在。而且现在第二周也过去一大半了，就按两周来写。
本（两）周总结：

  1. Todoist里的每日任务没有清空完
    任务有点多，外加拖延症发作，导致每天的任务都没有做完堆积到下一天。需要重新整理，合理规划，另外克服拖延症。
  2. 毕业设计没有头绪(到现在有了一点点头绪)
    主要没有学过信号相关的课程，对于图像信号处理一点都不懂。。这个礼拜谷歌了傅里叶变换和小波变换，但是博客上讲的都很散，公式也不多，感觉还是系统地看书比较好。这周大致了解了基本概念，另外看了一些比较常用的滤波算法，虽然不是很懂怎么实现的。
  3. 实习摸鱼
    想把dashbase那边的project写个测试收尾但是一直没有动。这周需要赶紧写一下测试。
  4. 学习摸鱼
    需要开始AI的学习之路，但还没有开动。以及要参加kaggle的训练赛。计划提上日程
    那么本周做了什么呢。。
    每天复习200个单词，这点坚持了。然后偶尔看CNN和VOA的新闻，未达到预期频率。大致了解频域和时频域变换。看了维纳滤波。似乎就没有做别的了。
    下周需要提高效率，再不搞毕设就有点凉了。。
